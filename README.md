# Weather App

Welcome to my Weather App! This project is my first major step into web development, with a primary focus on learning and applying CSS, React, and JavaScript. Throughout the process, I leveraged AI assistance to help guide me, as I am still building my knowledge and skills in these technologies.

## Features

- **Current Weather Data:** Fetches and displays real-time weather information for any city using an external weather API.
- **Responsive Design:** Built with CSS to ensure the app looks good,while being focused on mobile devices.
- **React Components:** Utilizes React for a modular, maintainable codebase.
- **API Integration:** Experiments with API keys and secure integration practices.
- **User Input:** Allows users to search for weather in different locations.

## Learning Journey

This project was a hands-on learning experience. I explored:

- **React Fundamentals:** Components, props, state management, and hooks.
- **JavaScript ES6+:** Modern syntax, async/await, and API requests.
- **CSS Styling:** Responsive layouts, flexbox, and custom styles.
- **API Usage:** Understanding endpoints, authentication, and handling responses/errors.

AI tools were used to assist with code suggestions, debugging, and understanding best practices. I also referenced documentation and community resources to deepen my understanding.

## Getting Started
1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/Weather-App.git
    cd Weather-App
    ```
2. **Install dependencies:**
    ```bash
    npm install
    ```
3. **Configure the API:**
    - The app uses an open-source weather API, so no API key is required.
    - If you wish to change the API endpoint, update the relevant configuration in the source code.

4. **Run the app:**
    ```bash
    npm start
    ```
    The app will open in your browser at `http://localhost:3000`.
